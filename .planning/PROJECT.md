# SidiCyber Tunisia

## Overview
AI-Powered Cybersecurity Training Simulator for Tunisian citizens. Hackathon project for "Defend the Digital Frontier" challenge.

## Tech Stack
- Next.js 16.1.6 (App Router, Turbopack)
- Vercel AI SDK + Azure OpenAI (GPT-4.1-mini)
- Tailwind CSS v4, Framer Motion
- TypeScript, Zod

## Core Features (Built)
- Landing page with hero, features, stats
- Phishing Simulator (8 scenarios, AI analysis)
- Scam Analyzer (AI-powered risk scoring)
- Legal Quiz (Tunisian cyber law)
- Trilingual i18n (AR/EN/FR)
- Dark cyberpunk theme
